# TAGS HTML

Documentação oficial - [html](https://developer.mozilla.org/pt-BR/docs/Web/HTML)

As **tags HTML** são elementos utilizados para estruturar e formatar o conteúdo de uma página web. Elas indicam ao navegador como exibir o conteúdo. Cada tag HTML é escrita entre colchetes angulares (`< >`). Por exemplo, `<p>` é uma tag para parágrafos, e `<h1>` é uma tag para cabeçalhos.

Aqui estão alguns exemplos de tags HTML comuns:

1. **`<html>`**: A tag raiz de um documento HTML. Tudo o que está dentro da página web deve estar dentro dessa tag.

   ```html
   <html>
     <!-- O conteúdo da página vai aqui -->
   </html>
   ```

2. **`<head>`**: Contém informações meta sobre o documento, como o título da página, links para estilos (CSS) e scripts (JavaScript).

   ```html
   <head>
     <title>Minha Página</title>
   </head>
   ```

3. **`<body>`**: Contém o conteúdo visível da página, como textos, imagens, links e outros elementos.

   ```html
   <body>
     <h1>Bem-vindo ao meu site!</h1>
     <p>Este é um parágrafo.</p>
   </body>
   ```

4. **`<h1>`, `<h2>`, `<h3>`, ... `<h6>`**: Tags de cabeçalhos, sendo `<h1>` o mais importante (geralmente o título principal da página) e `<h6>` o menos importante.

   ```html
   <h1>Cabeçalho Principal</h1>
   <h2>Subcabeçalho</h2>
   ```

5. **`<p>`**: Define um parágrafo de texto.

   ```html
   <p>Isso é um parágrafo de texto.</p>
   ```

6. **`<a>`**: Define um link. Atributos como `href` especificam o destino do link.

   ```html
   <a href="https://www.exemplo.com">Clique aqui</a>
   ```

7. **`<img>`**: Exibe uma imagem na página. Usa o atributo `src` para especificar o caminho da imagem.

   ```html
   <img src="imagem.jpg" alt="Descrição da imagem">
   ```

8. **`<ul>`, `<ol>`, `<li>`**: Usadas para listas. `<ul>` é para listas não ordenadas (com marcadores) e `<ol>` é para listas ordenadas (numeradas). Os itens da lista são definidos com a tag `<li>`.

   ```html
   <ul>
     <li>Item 1</li>
     <li>Item 2</li>
   </ul>
   ```

9. **`<section>`**: Define uma seção de conteúdo em um documento. Geralmente é usado para agrupar conteúdo tematicamente relacionado, como uma parte de um artigo ou um capítulo de um livro.
 ```html
   <section>
     <h1>Título 1</h1>
     <p>Item 2</p>
   </section>
   ```

Além disso, algumas tags podem ter **atributos** que modificam o comportamento ou estilo de um elemento. Por exemplo, a tag `<a>` usa o atributo `href` para definir o destino do link, enquanto a tag `<img>` usa `src` para especificar a localização da imagem.

Essas tags são os blocos de construção essenciais para a criação de páginas web!
